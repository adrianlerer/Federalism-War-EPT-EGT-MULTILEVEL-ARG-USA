## üß¨ MARCO TE√ìRICO: EPT + EGT MULTINIVEL

### **Extended Phenotype Theory (Dawkins 1982)**

**Aplicaci√≥n a federalismo**:

**Definici√≥n**: Provincia/estado como **extended phenotype** de coalici√≥n gobernante

**Mecanismos de replicaci√≥n**:
1. **Legislaci√≥n provincial**: Leyes como "genes" culturales (memes)
2. **Instituciones locales**: ONGs, agencias como "√≥rganos" del fenotipo
3. **Manipulaci√≥n del ambiente federal**: Captura de recursos como "construcci√≥n de nicho"

**Selecci√≥n**:
- **Provincias exitosas**: Aquellas que capturan m√°s recursos federales
- **Replicaci√≥n**: Otras provincias copian estrategias exitosas
- **Evoluci√≥n**: Sistema tiende hacia equilibrio extractivo

---

### **Multilevel Game Theory (Boyd & Richerson 2002)**

**Niveles de selecci√≥n**:

**Nivel 1: Intra-provincial** (within-group)
- Competencia entre partidos locales por control provincial
- Selecci√≥n favorece estrategias clientelistas

**Nivel 2: Inter-provincial** (between-group)
- Competencia entre provincias por recursos federales
- Selecci√≥n favorece estrategias extractivas

**Nivel 3: Federal** (meta-level)
- Gobierno federal intenta coordinar, pero carece de enforcement
- Selecci√≥n de instituciones federales (d√©biles vs fuertes)

**Predicci√≥n te√≥rica**:
- Si **selecci√≥n intra-provincial** > **selecci√≥n federal** ‚Üí estrategias extractivas dominan
- Si **selecci√≥n federal** > **selecci√≥n intra-provincial** ‚Üí cooperaci√≥n se mantiene

---

### **Replicator Dynamics**

**Poblaci√≥n**: N provincias/estados  
**Estrategias**: C (Cooperate), D (Defect/Extract)  
**Frecuencias**: p (proporci√≥n C), 1-p (proporci√≥n D)

**Payoffs**:
```
E(C, C) = R (reward cooperaci√≥n mutua)
E(C, D) = S (sucker payoff, provincia cooperativa es explotada)
E(D, C) = T (temptation, provincia extractiva explota cooperativas)
E(D, D) = P (punishment, todas extractivas, colapso federal)

Condici√≥n para cooperaci√≥n: R > T y R > P
Condici√≥n para defecci√≥n: T > R y P > S (Prisoner's Dilemma)
```

**Din√°mica replicadora**:
```
dp/dt = p(1-p)[E(C) - E(D)]

Si E(D) > E(C) ‚Üí p ‚Üí 0 (extinci√≥n cooperaci√≥n)
Si E(C) > E(D) ‚Üí p ‚Üí 1 (extinci√≥n extracci√≥n)
```

**Aplicaci√≥n emp√≠rica**:
- **Argentina**: p ‚Üí 0 (cooperaci√≥n extinta, todas las provincias extraen)
- **EEUU**: p ‚Üí 0.3-0.4 (cooperaci√≥n en decline, ~15-20 sanctuary states)
